# API Workflow Orchestration Example

This example demonstrates API orchestration by simulating a complex e-commerce order workflow. It coordinates multiple 'API calls' (stock check, payment processing, order creation, email confirmation) in a specific sequence. The code highlights sequential execution, conditional logic, and robust error handling, including compensation actions like attempting a refund if a later step fails, which are key aspects of API orchestration.

## Language

`python`

## How to Run

1. Save the code as `main.py`.
2. Open your terminal or command prompt.
3. Run the script using Python: `python main.py`
